# 🎯 PRONTO! Agora Faça Isso:

## 1️⃣ Criar Repositório GitHub (1 minuto)

```
Vá para: https://github.com/new

Nome: vamos-la-tickets
Criar

Copiar URL (ex: https://github.com/seu-user/vamos-la-tickets.git)
```

---

## 2️⃣ Copiar Esses Comandos (2 minutos)

Abra terminal em `/home/justino/validaçao tickets` e execute:

```bash
# Substituir a URL com a sua!!!
git remote add origin https://github.com/seu-user/vamos-la-tickets.git

git add -A
git commit -m "Vamos Lá Tickets v1.0"
git branch -M main
git push -u origin main
```

---

## 3️⃣ Aguardar 20 Minutos ⏳

GitHub compila APK automaticamente!

---

## 4️⃣ Download APK (1 minuto)

```
Vá para: https://github.com/seu-user/vamos-la-tickets/actions

Status: ✅ SUCCESS (verde)

Clique em "Build Flutter APK"
Scroll para "Artifacts"
Clique em "flutter-app-apk"
Download: app-release.apk (~17 MB)
```

---

## 5️⃣ Instalar no Telefone (2 minutos)

Telefone conectado via USB + USB Debugging ativado

```bash
adb install -r ~/Downloads/app-release.apk
```

---

## ✅ PRONTO! 🎉

App instalado no seu telefone!
