## 🔄 FLUXO DE USO

```
1. QR escaneado
   ↓
2. App valida JSON localmente
   ↓
3. App envia para Servidor (LAN)
   ↓
4. Servidor valida assinatura HMAC
   ↓
5. Servidor busca ticket no DB
   ↓
6. Servidor marca como usado (lock atômico)
   ↓
7. Servidor responde (valid/already_used/invalid)
   ↓
8. App mostra resultado (verde/vermelho/amarelo)
   ↓
9. App emite som (beep)
```

## 💡 INFORMAÇÕES IMPORTANTES

### IP Fixo (IMPORTANTE!)
Atualizar em: `mobile_app/lib/main.dart`
```dart
const String SERVER_URL = 'http://192.168.1.100:8000';
```
Trocar `192.168.1.100` pelo IP do seu servidor!

### SECRET_KEY (Segurança)
Em produção, alterar em: `local_server/.env`
```
SECRET_KEY=sua-chave-super-secreta-aqui
```

### Porta (Se necessário)
Em: `local_server/.env`
```
PORT=8000
```
