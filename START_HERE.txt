⏱️ TIMELINE DE USO
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1️⃣  PRÉ-EVENTO (1 semana antes)
    ├─ [ ] Ler README.md (5 min)
    ├─ [ ] Seguir INSTALLATION.md (30 min)
    ├─ [ ] Testar servidor (10 min)
    ├─ [ ] Gerar APK (5 min)
    ├─ [ ] Instalar em 3-6 devices (15 min)
    └─ [ ] Teste completo (30 min)
    
    ⏳ Total: ~1.5 horas

2️⃣  DIA DO EVENTO (2 horas antes)
    ├─ [ ] Servidor iniciado
    ├─ [ ] Criar evento (POST /events)
    ├─ [ ] Sincronizar tickets (POST /sync-tickets)
    ├─ [ ] Testes de múltiplos scanners
    ├─ [ ] Verificar internet vs LAN
    └─ [ ] Equipe pronta
    
    ⏳ Total: ~30 min

3️⃣  DURANTE EVENTO
    ├─ [ ] Scanners ativos
    ├─ [ ] Monitor stats (/events/:id/stats)
    ├─ [ ] Verificar logs
    └─ [ ] Suporte se necessário
    
    ⏳ Contínuo

4️⃣  PÓS-EVENTO
    ├─ [ ] Sincronizar logs com backend (opcional)
    ├─ [ ] Fazer backup database
    ├─ [ ] Gerar relatório
    └─ [ ] Arquivar dados
    
    ⏳ Total: ~15 min
