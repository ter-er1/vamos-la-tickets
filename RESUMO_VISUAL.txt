PASSO 1: Ler README.md (5 minutos)
─────────────────────────────────
$ cat README.md
└─ Entender a arquitetura geral

PASSO 2: Seguir INSTALLATION.md (30 minutos)
─────────────────────────────────
$ cat INSTALLATION.md
└─ Instruções passo a passo

PASSO 3: Instalar Servidor (2 minutos)
─────────────────────────────────
$ cd local_server
$ npm install
$ npm start

PASSO 4: Testar Servidor (1 minuto)
─────────────────────────────────
$ cd local_server
$ node test-api.js

PASSO 5: Gerar QRs (2 minutos - Opcional)
─────────────────────────────────
$ python3 generate_qr_codes.py

PASSO 6: Build APK (5 minutos - Opcional)
─────────────────────────────────
$ cd mobile_app
$ flutter pub get
$ flutter build apk --release

PASSO 7: Usar! (∞ minutos 😄)
─────────────────────────────────
└─ Instalar APK em 3-6 devices
└─ Fazer login
└─ Escanear QR codes
└─ Ver resultados!
