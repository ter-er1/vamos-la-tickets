SUAS PERGUNTAS? RESPOSTAS DIRETAS:
────────────────────────────────────────────────────────────────────────────────

P1: "Agora como vou rodar o servidor web?"
R1: Digite isto no terminal:
    
    cd "validaçao tickets/local_server"
    npm install
    npm start
    
    Pronto! Servidor rodando.

P2: "E onde está o aplicativo para instalar nos telefones?"
R2: Em código. Compile com:
    
    cd "validaçao tickets/mobile_app"
    flutter build apk --release
    
    APK fica em: build/app/outputs/flutter-app-release.apk


ETAPAS PARA FUNCIONAR (20 minutos):
════════════════════════════════════════════════════════════════════════════════

1. SERVIDOR (2 min)
   ├─ cd validaçao\ tickets/local_server
   ├─ npm install
   ├─ npm start
   └─ ✅ Deixe aberto

2. APP (10 min)
   ├─ Editar: mobile_app/lib/main.dart (linha 16, atualizar IP)
   ├─ cd validaçao\ tickets/mobile_app
   ├─ flutter pub get
   ├─ flutter build apk --release
   └─ ✅ APK pronto

3. INSTALAR (5 min)
   ├─ flutter install  (ou copiar APK manualmente)
   └─ ✅ Pronto!
